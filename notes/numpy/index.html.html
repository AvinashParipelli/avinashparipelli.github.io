NUMPY

What is NumPy?

NumPy is a Python library used for working with arrays.

importing numpy

The standard way to  

import NumPy:

import numpy as np

NumPy is usually imported under the np alias.

Creating an array

We can create a NumPy ndarray object by using the  

array()

function.

arr = np.array([1, 2, 3, 4, 5])

dimensional Numpy

Numpy have many dimensional arrays like 0-D , 1-D, 2-D, 3-D

import numpy as np

arr = np.array(42) #0-D

arr = np.array([1, 2, 3, 4, 5]) #1-D

arr = np.array([[1, 2, 3], [4, 5, 6]]) #2-D

arr = np.array([[[1, 2, 3], [4, 5, 6]], [[1, 2, 3], [4, 5, 6]]]) #3-D
